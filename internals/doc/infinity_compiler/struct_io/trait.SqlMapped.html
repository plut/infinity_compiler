fn get_field(&self, i: usize) -> ToSqlOutput<'_>

Returns the i-th field of the resource, wrapped in a [rusqlite::types::ToSlqOutput] for trivial insertion into a SQL statement. NOTE: rusqlite does not allow pushing Result<ToSqlOutput> values back into a [rusqlite::ParamsFromIter] iterator, so we need to unwrap those values. Since we do so from a controlled struct, this should not panic; however, it would be better to be able to propagate errors.
