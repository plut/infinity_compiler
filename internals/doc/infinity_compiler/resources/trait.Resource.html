Expand description

The trait shared by all game resources. All the required functions are produced by the Resource derive macro from the list of sub-resources for this resource.

    &self,
    db: &Connection,
    node: &mut Self::StatementNode<'_>,
    resref: Self::Primary
) -> Result<usize>

Inserts a top-level resource. This differs from subresources in that the primary key (resref) is known.

This is the entry point called when populating the database; this function is not called recursively.

    db: &Connection,
    node: &mut Self::StatementNode<'_>,
    parent: impl ToSql,
    position: usize
) -> Result<()>

Inserts a sub-resource.

This differs from a top-level resource in that the primary key (rowid) is not known yet; instead it is derived from last_insert_rowid.

This function is mutually recursive with Self::insert_subresources (derived from macro).
